This is my project for Infoblox Assessment. The technologies that have been used are given below:

  • Django 3.1
  • Django Rest Framework 3.11.1
  • Postgresql with Psycopg2 package

Eventhough, I have heard about the Django Rest Framework, I did not have an opportunity to work on it. I have worked on Django framework before, for few projects. Please contact me, in case of any queries concerning the project. The setup I have made with Django Rest Framework is pretty basic one with the help of Django Rest Framework tutorial. The pagination used is Page number pagination hence, for viewing all the data, you have to append ?page[number] for all data

Filtering has been applied to the /api/all page. Give : /api/all?enabled=True

The useful links are given below for your reference :

  • List all data
  • Add new data
  • Get detail of a user with id : /api/[id]

The command line executions that have been implemented are :

  • python3 manage.py add_data [path to JSON file]
  • python3 manage.py read_data [id]
