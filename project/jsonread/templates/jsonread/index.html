Packages and Tools Used

  • Django Framework 3.1.1
  • Django Rest Framework 3.11.1
  • Postgresql with psycopg2 package
  • Bootstrap 4

Command Line Executions

  • python3 manage.py add_data [path_to_json_file]

    The command to insert the data from the JSON file into the models. Each and every key in the sample JSON has been mapped to the appropriate column in the table.

  • python3 manage.py read_data [id]

    The command is used to get you the first and last name of the id that is provided.

Setting up the project for testing

  1. Create a Database info in your postgres
  2. Create a user sample_user with password infoblox123 in your postgres server.
  3. Run python3 manage.py migrate for importing the models into your postgres info database
  4. Create a superuser for you to control the application python3 manage.py createsuperuser
  5. Run the application in your local system by running the command python3 manage.py runserver
